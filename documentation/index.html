Get Started

First, download your fresh update version from Codecanyon. You will get a zip file called main.zip extract this file in your desire folder. After extract you will see files and folders like below image.

Now go to inside the extracted folder and browse application -> config folder

to configure database and others. Find config.php, Database.php. and Paypal.php. files and edit with your editor. Now change following options with your proper value. Please be extra careful to edit the config.php and Paypal.php. files. Remember Paypal only works when your selected currency is supported with Paypal.

    config.php

  • $config['base_url'] = 'http://yoursite.com/';

    • Please insert your site's right URL or localhost path in base_url, otherwise noting will work.

    database.php

  • 'database' => 'your database'
  • 'username' => 'your database user name',
  • 'password' => 'your database password',

    • paypal.php (for both paypal & stripe)

  • $config['client_id'] = 'your paypal client id'
  • $config['secret'] = 'your paypal secret key'
  • 'mode' => 'live', (change paypal mode to live)
  • $config['stripe_apikey'] = 'your stripe secret api key'

    • zal_config.php (for both SMS & Email)

Now go to your cpanel area and select file manager. Inside file manager select public_html from left side menu. Now click upload button from top bar and upload all the files and folders from your Zal application.

Now go to your cpanel area again and select MySQL Database. Inside MySQL database create a new database and a new user. Combine new database and new user together. Remember your database name, user name and password for later. Go to your Phpmyadmin panel and select your database which you created recently. Click on import button to import database from application. Browse inside Zal folder and import database-final.sql and import. All data will be imported and you are almost done.

You are done and ready to use your ISP application. Browse http://yoursite.com/ to see your site. You will see your applicaiton login page like below image.

Dashboard -> User

User module is for listing and managing users/clients of your business. You can add, edit, view and delete. Also you can browse user's bills by year in their profile page.

Dashboard -> Bills & Invoices

This module will helps you to generate auto bills and invoices for all of your clients. Just select create date and due date then click on generate now button. You can also generate individual bills and invoices maually in below.

Browse bills as well by month and year in this same module. In browse bills section you will see Padi users and Unpaid users with their invoices.

Dashboard -> Package

Package module is for managing packages with extra 5 costs. Those extra 5 cost's field is for your extra service charge like yearly service charge etc. All costs will automatically calculated in every bills and invoice. So be carefull to insert cost value.

Dashboard -> Invoice

Invoice module is for managing all invoices. You can create a new invoice in Add Bills module. You can make invoice status paid or unpaid manually. You can generate PDF file of invoice to send over email. You can print invoice and User can pay invoice bills by Paypal & Stripe. All invoice amount is based on your default selected currency.

Dashboard -> Payment

All Payments are shown here. All those payments which made by paypal, stripe and manually will added here. You can brose payments by month and year. Payment statistics also available here.

Right now Zal supports Paypal & Stripe but in next release we will integrate other pyament methods. So, to use Paypal & Stripe properly, you have to insert your Paypal & Stripe credential in paypal.php file, which is located applicaiton -> config folder. To get your Paypal credential go to Paypal Developer and for Stripe credential go to Stripe Developer and login in with your Paypal/Stripe account. From the left site click my apps and credentials. Create new apps with your ISP name and get credentials, don't forget to seletc transaciton mode "sandbox" or "live" at the top.

Right now PAYPAL_MODE is sandbox, which is for testing purpose. Insert PAYPAL_MODE to live if you want real transacitons, check paypal.php file. Please read Paypal documentation properly before using Paypal API. Also, keep in mind Paypal is restricted in many countries, so please check first if Paypal is supported in your country.

You can understand those income which are came from payments by their title ex : Stripe Payment or Paypal Payment.

Dashboard -> PPPoE & Hotspot

Requirements For Mikrotik

  • 1. RouterOS Minimum Version 3
  • 2. API Service Enable
  • 3. Firewall Open For API 8728/tcp & 8729/tcp (For SSL)
  • 4. PHP stream_socket_client() Enable in server/host
  • 5. Open Port in Your Hosting 8728 & 8729 (ASK Your Hosting Provider)
  • 6. Enable API Service in User Profile of Mikrotik

Go to setting page and insert your public Mikrotik IP address and User Name and Password of your Mikrotik router. Insert only IP address in IP address field in setting page, no port is required. Now go to Logs page to see if your device is connected. If you see your router logs then its connected successfully.

Zal supported with Mikrotik, you can do some basic tasks with Zal like controlling PPPoE & Hotspot Users. You can create new PPPoE and Hotspot Users. You can enable/disable PPPoE and Hotspot User. Connect Portal Users with PPPoE and Hotspot User's secret. Manage Users Bill and others.

Zal only support Mikrotik router/routerOS at this moment but we will add others soon. To use Mikrotik enable Mikrotik API in your Mikrotik Router. Be sure that your router is setup correctly to listen API calls. Set your Mikrotik IP address, user and password in settings page.

Disable PPPoE and Hotspot User from User profile page. You can see PPPoE Disable and Hotspot Disable button under email change button.
